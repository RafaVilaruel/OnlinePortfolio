Description

Java RPG game

A Java RPG game using pure JavaFX.

"Got emotional in the end. Felt like i was reading an epic adventure story all the time."
 "Loved the main caracther, the story, the sounds, im soooo ready for the sequence."

Iaycha is an adventure game with a 100% original story created all by myself. Everything started with a simple study case of an engine. The project kept growing and got out of hand. Fully complete, the game has an begginning to end story. With a lot of paths you decide yourself taking, changing the story and creating your very own!

The game has more than 20 original caracthers, musics, environment, a save/load system and an engine ready for patchs and sequences in the future.
